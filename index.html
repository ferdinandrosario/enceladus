Installation

Add this line to your application's Gemfile:

gem 'enceladus'

And then execute:

$ bundle

Or install it yourself as:

$ gem install enceladus

Usage

You first need to sign up to The Movie Database and then request a new api key.

Have a look at some usage examples and check out the docs too.
